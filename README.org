* usage
  | *key(s)*             | *action*                    |
  | SPACE             | drop block                  |
  | LEFT/RIGHT        | move left/right             |
  | UP                | rotate clockwise            |
  | 1234qwersd        | move to ith column and drop |
  | Ctrl+C (any key)  | switch human/ai player      |
  | Ctrl+C Ctrl+C     | exit                        |
